We built a Library Management System GUI using:
•	Tkinter → for the Graphical User Interface (windows, buttons, inputs) 
•	SQLite for database management (functions like  (add_book, view_books, delete_book, update_books) and (add_member, view_members, delete_member and update_member) )
The system allows users to manage the following using CRUD operations:
•	Books 
•	Library members

Objectives:
The objectives of this project are:
•	To design a user-friendly GUI
•	To implement CRUD operations
•	To store data using SQLite 
•	To connect the GUI with the database
•	Lastly, to apply error handling and validation

To achieve the above objectives, the following technologies were used:
•	Python: Main programming language
•	Tkinter: GUI development
•	SQLite: Database storage

Additional Information:
Tkinter allows us to create windows, buttons, and forms, SQLite helps to store data locally in a file.

System Structure:
This project is divided into multiple modules:
•	Main.py: Main application window
•	db.py: Database connection and table creation
•	books.py: Book interface and CRUD operations
•	members.py: Member interface and CRUD operations

Additional information:
This system structure improves code readability and maintainability.
